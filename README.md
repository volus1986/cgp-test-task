Mini-builder with the ability to add, remove, edit, and rearrange blocks:

Left toolbar: blocks can be dragged or added to the workspace with a click.

Workspace: when you click a block it becomes active, showing a toolbar with buttons and enabling editing.

Right section: displays the result of the configuration.
